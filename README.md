# Stroke-prediction--MACHINE LEARNING
## Project Overview
This project focuses on predicting the risk of strock using Machine Learning techniques.The model is trained on healthcare data and aims to identify whether a patient is likely to suffer from a strocke based on various medical and demographical factors.

## Objectives:
 *Analyze healthcare data and understand important features,
 *Perform data preprocessing and exploaratory data analysis(EDA),
 *Remove outlier using the IQR method,
 *Train multiple machine learning models and compare their performance,
 *predict stroke occurence with improved accuracy.

 ## Dataset:
  Gender,Age,Hypertension,Heart Disease,Marital Status, Work Type, Residence Type, Average Gulcose Level,BMI,Smoking Status,stroke(Target variable)

  ## Project Workflow:
  Data collection-->Data Cleaning and Preprocessing-->Data Understanding-->Exploratory Data Analysis(EDA)-->Outlier Detection and Removal(IQR Method)-->Features Selection and Data Splitting-->Model Training and Evalution-->Performance Comparison-->Stroke Prediction

  ## Machine Learning Models Used
  Logistic Regression,Decision Tree Classifier, Random Forest Classifier ,K-Nearest Neighbors(KNN),Support Vector Machine(SVM),Gradient Boosting Classifier,AdaBoost Classifier,XGBoost classifier
